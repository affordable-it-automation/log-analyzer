# Log File Analyzer (Python)

## What it does
This script scans log files, detects errors, and outputs a clean summary for easier troubleshooting.

## Example
Input: raw_log.txt  
Output: error_summary.txt

## Why it's useful
Manually reading logs takes time. This script quickly finds errors and organizes them into a simple, readable format.

## How to run
python log_analyzer.py
